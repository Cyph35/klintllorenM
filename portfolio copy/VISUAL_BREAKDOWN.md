## Responsive Breakpoints

### Desktop (1200px+)
- Full container width
- 2-column skill grids
- Optimal reading measure (60-70 chars/line)
- Generous spacing and padding
- Full hover effects

### Tablet (768px - 1199px)
- 95% container width with padding
- Single-column skill grids
- Adjusted font sizes
- Medium spacing
- Touch-optimized buttons

### Mobile (480px - 767px)
- Full-width with side padding
- Vertical stacking throughout
- Optimized font hierarchy
- Compact spacing
- 44px+ button targets

### Small Mobile (360px - 479px)
- Minimal side padding (8-16px)
- Further reduced spacing
- Tight but readable typography
- Maximum tap target size
- No unnecessary whitespace
